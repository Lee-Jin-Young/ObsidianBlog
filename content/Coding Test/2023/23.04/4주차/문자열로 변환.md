### 문제

###### 문제 설명

정수 `n`이 주어질 때, `n`을 문자열로 변환하여 return하도록 solution 함수를 완성해주세요.

##### 제한사항

- 1 ≤ `n` ≤ 10000

---

### 나의 풀이

```java
class Solution {
    public String solution(int n) {
        return String.valueOf(n);
    }
}
```
