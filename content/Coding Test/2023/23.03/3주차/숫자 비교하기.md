### 문제

###### 문제 설명

정수 `num1`과 `num2`가 매개변수로 주어집니다. 두 수가 같으면 1 다르면 -1을 retrun하도록 solution 함수를 완성해주세요.

##### 제한사항

- 0 ≤ `num1` ≤ 10,000
- 0 ≤ `num2` ≤ 10,000

---

### 나의 풀이

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        
        if (num1 == num2) {
            answer = 1;
        } else {
            answer = -1;
        }
        
        
        return answer;
    }
}
```

### 다른 사람의 풀이

```java
class Solution {
    public int solution(int num1, int num2) {
        return (num1 == num2) ? 1 : -1;
    }
}
```
