### 문제

###### 문제 설명

머쓱이는 40살인 선생님이 몇 년도에 태어났는지 궁금해졌습니다. 나이 `age`가 주어질 때, 2022년을 기준 출생 연도를 return 하는 solution 함수를 완성해주세요.

##### 제한사항

- 0 < age ≤ 120
- 나이는 태어난 연도에 1살이며 1년마다 1씩 증가합니다.

---

### 나의 풀이

```java
class Solution {
    public int solution(int age) {
        int answer = 0;
        
        answer = 2022-age+1;
        
        return answer;
    }
}
```
