### 문제

###### 문제 설명

머쓱이네 피자가게는 피자를 일곱 조각으로 잘라 줍니다. 피자를 나눠먹을 사람의 수 `n`이 주어질 때, 모든 사람이 피자를 한 조각 이상 먹기 위해 필요한 피자의 수를 return 하는 solution 함수를 완성해보세요.

##### 제한사항

- 1 ≤ `n` ≤ 100

---

### 나의 풀이

```java
class Solution {
    public int solution(int n) {
        int answer = 0;
        
        if(n%7==0) answer = n/7;
        else answer = n/7+1;
        
        return answer;
    }
}
```

| n       | n / 7 | 필요한 피자 |
| ------- | ----- | ----------- |
| 1 ~ 6   | 0     | 1           |
| 7       | 1     | 1           |
| 8 ~ 13  | 1     | 2           |
| 14      | 2     | 2           |
| 15 ~ 20 | 2     | 3           |
| 21      | 3     | 3           | 
### 다른 사람의 풀이

```java
class Solution {
    public int solution(int n) {
        return (n + 6) / 7;
    }
}
```

| n       | n+6     | (n+6) / 7 |
| ------- | ------- | --------- |
| 1 ~ 7   | 7 ~ 13  | 1         |
| 8 ~ 14  | 14 ~ 20 | 2         |
| 15 ~ 21 | 21 ~ 27 | 3         |
